# Lexipane

**Мгновенный перевод и словарь прямо на экране — без переключения окон.**

Наведи, выдели или захвати любой текст на экране — игру, PDF, статью — и получи перевод, грамматику и произношение в аккуратном всплывающем окне. Одной клавишей сохрани слово в Anki.

---

## Что умеет

- **Захват экрана** — выдели область или наведи на слово в любом окне
- **OCR** — переводит текст из игр, PDF, изображений
- **Богатый попап** — лемма · IPA · часть речи · частота · словарь · озвучка
- **Сохранение в Anki** — карточки с одной клавиши
- **Профили** — разные настройки для разных приложений

## Системные требования
