Changelog

Release history. Download links live on the GitHub Releases page.

v0.4.x · June 2026

  • Universal desktop overlay positioning for articles, books, games, PDFs and more.
  • Free / Pro plan model with license key activation foundation.
  • Plan settings tab: activate key, view credits and upgrade link.
  • Design system foundation: tokens, theme manager and reusable components.
  • Public site, legal pages, FAQ, guides and release download surface.
  • Live subtitles work: system audio capture, local Whisper, translation and overlay UX improvements.

v0.3.x · May-June 2026

  • Renamed from GameLingo to Lexipane.
  • OCR improvements: Google Lens integration and macOS native fallback.
  • Rich word popup: lemma, IPA, POS, frequency band and context sentence.
  • NoAI provider, local dictionary support and Anki export improvements.
  • Windows portable build and macOS DMG build via PyInstaller.
