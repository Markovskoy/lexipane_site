FAQ

Common questions about Lexipane. More questions? Ask in the Telegram channel.

What does Lexipane do?

Lexipane is a desktop overlay that translates and explains text from any window on your screen - articles, books, games, PDFs, or subtitles - without switching apps. You can save words to Anki with one key.

Does it work with any application?

Yes. Lexipane reads the screen directly using OCR, so it works with browsers, games, PDF viewers, terminals, IDEs, and most desktop windows.

Is an internet connection required?

Basic capture, OCR, dictionary lookup and local history can work offline with installed data. Online translation and AI explanations need a provider connection unless you use a fully local endpoint.

Is my screen content sent to a server?

Only the text you capture is sent, and only to the provider mode you choose. Lexipane Cloud goes through the Lexipane backend; your own key or custom endpoint goes directly to that provider.

What is Free vs Pro?

Free includes screen capture, OCR, dictionary translation and Anki export. Pro is planned for Lexipane Cloud credits, AI explanations, Game Freeze and billing-backed license activation.

Can I use Lexipane without Anki?

Yes. Anki is the primary integration, but clipboard, markdown and local history workflows can still be useful.

What happens when AI credits run out?

Lexipane Cloud pauses until renewal or top-up. You can continue in Without AI mode or switch to your own API key/custom endpoint.

FAQ

Частые вопросы о Lexipane. Остальные вопросы можно задать в Telegram.

Что делает Lexipane?

Lexipane - desktop overlay, который переводит и объясняет текст из любого окна на экране: статьи, книги, игры, PDF или субтитры. Переключаться между приложениями не нужно. Слова можно сохранять в Anki одной клавишей.

Работает ли с любым приложением?

Да. Lexipane читает экран через OCR, поэтому работает с browsers, games, PDF viewers, terminals, IDEs и большинством desktop windows.

Нужен ли интернет?

Базовый capture, OCR, dictionary lookup и local history могут работать offline с установленными данными. Online translation и AI explanations требуют provider connection, если вы не используете полностью локальный endpoint.

Отправляется ли содержимое экрана на сервер?

Отправляется только тот текст, который вы захватили, и только в выбранном provider mode. Lexipane Cloud идет через backend Lexipane; свой key или custom endpoint идет напрямую к этому provider.

Чем отличаются Free и Pro?

Free включает screen capture, OCR, dictionary translation и Anki export. Pro планируется для Lexipane Cloud credits, AI explanations, Game Freeze и license activation через billing.

Можно ли без Anki?

Да. Anki - основная интеграция, но clipboard, markdown и local history тоже могут быть полезны.

Что если AI-кредиты закончились?

Lexipane Cloud ставится на паузу до продления или top-up. Можно продолжать в Without AI mode или переключиться на свой API key/custom endpoint.
