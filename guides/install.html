Install Lexipane

WindowsmacOSInstall · permissions · first run

Windows

Windows 10 / 11 · 64-bit

  1. Download the installer from the main page and save Lexipane.exe.
  2. Run the installer. If SmartScreen warns you, choose More info -> Run anyway.
  3. Finish the installer and launch Lexipane from Start Menu or Desktop.
  4. On first run, choose your languages and translation mode.
Antivirus warning: unsigned or fresh installers can trigger false positives. Lexipane does not contain malware; add the install folder to exclusions if needed.

macOS

macOS 13 Ventura or newer

  1. Download the DMG from the main page.
  2. Open it and drag Lexipane.app into Applications.
  3. For the first launch, right-click the app in Finder and choose Open.
  4. Approve Accessibility, Input Monitoring and Screen Recording permissions.
If Gatekeeper blocks the app: run xattr -dr com.apple.quarantine /Applications/Lexipane.app, then launch again from Finder.

First capture

Press the capture hotkey, select a screen area with text, and Lexipane will show translation in an overlay. Then adjust languages and provider mode in Settings.

Установка Lexipane

WindowsmacOSУстановка · разрешения · первый запуск

Windows

Windows 10 / 11 · 64-bit

  1. Скачайте установщик на главной странице и сохраните Lexipane.exe.
  2. Запустите установщик. Если SmartScreen показывает предупреждение, выберите Подробнее -> Выполнить в любом случае.
  3. Завершите установку и запустите Lexipane из меню Пуск или с рабочего стола.
  4. При первом запуске выберите языки и режим перевода.
Антивирусы: unsigned или свежие установщики иногда дают false positive. Lexipane не содержит вредоносного кода; при необходимости добавьте папку установки в исключения.

macOS

macOS 13 Ventura и новее

  1. Скачайте DMG на главной странице.
  2. Откройте его и перетащите Lexipane.app в Applications.
  3. Для первого запуска найдите app в Finder, нажмите правой кнопкой и выберите Open.
  4. Разрешите Accessibility, Input Monitoring и Screen Recording.
Если Gatekeeper блокирует приложение: выполните xattr -dr com.apple.quarantine /Applications/Lexipane.app, затем снова запустите через Finder.

Первый захват

Нажмите hotkey захвата, выделите область с текстом, и Lexipane покажет перевод в overlay. Потом настройте языки и provider mode в Settings.
