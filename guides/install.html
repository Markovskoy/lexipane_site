Установка Lexipane

Windows macOS Установка · Разрешения · Первый запуск

Windows

Windows 10 / 11 · 64-bit

  1. Скачайте установщик — нажмите кнопку «Скачать для Windows» на главной странице, сохраните файл Lexipane.exe.
  2. Запустите установщик. Если Windows SmartScreen показывает предупреждение — нажмите «Подробнее» → «Выполнить в любом случае».
  3. Следуйте шагам установщика. После завершения запустите Lexipane из меню Пуск или с рабочего стола.
  4. При первом запуске пройдите быструю настройку — выберите языки и режим перевода.
Антивирусы: некоторые антивирусы могут ложно срабатывать на установщик. Это ложное срабатывание — Lexipane не содержит вредоносного кода. Добавьте папку установки в исключения, если возникают предупреждения.

macOS

macOS 13 Ventura и новее

  1. Скачайте DMG — нажмите кнопку «Скачать для macOS» на главной странице.
  2. Откройте скачанный файл .dmg. Перетащите Lexipane.app в папку Applications.
  3. Откройте Terminal (Finder → Программы → Утилиты → Terminal) и выполните команду:

    xattr -dr com.apple.quarantine /Applications/Lexipane.app

    Это снимает карантинный флаг macOS, после чего приложение откроется без предупреждений.
  4. macOS попросит три разрешения — нажмите «OK» на каждое:
    • Специальные возможности (Accessibility)
    • Мониторинг ввода (Input Monitoring)
    • Запись экрана (Screen Recording)
Разрешения запрашиваются один раз — при последующих обновлениях повторно давать их не нужно. Если разрешение не было выдано при первом запуске, откройте Системные настройки → Конфиденциальность и безопасность и включите там вручную.

Первый запуск

После установки нажмите хоткей захвата (по умолчанию Ctrl+Shift+F на Windows / ⌃⇧F на Mac), выделите область с текстом — Lexipane покажет перевод во всплывающем окне.

В настройках выберите языки перевода и источник переводов. Для режима с AI-переводом потребуется ключ API или подписка Pro.
