AI modes in Lexipane

ProOpenAIWithout AIProvider settings · credits · local fallback

Lexipane has three AI paths. Lexipane Cloud uses Pro credits through the Lexipane backend. OpenAI with your own key connects directly from your computer and is billed by OpenAI to your account. Without AI uses dictionaries and translator fallback without an LLM provider.

Provider settings in the desktop app.

Lexipane Cloud

  • Best for: Pro users who want AI explanations without managing an OpenAI key.
  • Credits: the monthly allowance is counted on the backend by actual model usage. When credits run out, Cloud AI pauses until renewal or top-up.
  • Privacy: selected text is sent to the Lexipane backend, then to OpenAI. The desktop app never contains the master OpenAI key.

OpenAI with your own key

  • Best for: unlimited personal usage under your own OpenAI billing limits.
  • Billing: requests go directly from your computer to OpenAI and do not spend Lexipane Cloud credits.
  • Connection: your network must be able to reach OpenAI servers. If OpenAI is unavailable in your region, use a VPN or switch to Without AI.

Without AI mode

  • Best for: quick translation, offline dictionary lookups and basic Anki cards.
  • Online fallback: can use Google Translate unofficial endpoint when enabled.
  • Offline-only: disables web requests, but quality depends entirely on installed dictionary packs.

Credit rule

Lexipane Cloud credits are enforced on the server. The desktop app never stores the master OpenAI key and never creates OpenAI keys by itself. If the backend or OpenAI is unavailable, Lexipane shows a connection message and you can retry later, enable VPN, switch to your own key, or use Without AI.

AI-режимы в Lexipane

ProOpenAIWithout AIПровайдеры · кредиты · локальный fallback

В Lexipane есть три пути. Lexipane Cloud использует Pro-кредиты через backend Lexipane. OpenAI со своим ключом подключается напрямую с вашего компьютера и оплачивается в вашем аккаунте OpenAI. Without AI использует словари и переводчик без LLM-провайдера.

Настройки провайдера в desktop-приложении.

Lexipane Cloud

  • Для кого: Pro-пользователи, которым нужны AI-объяснения без настройки собственного OpenAI key.
  • Кредиты: месячный лимит считается на backend по фактическому расходу модели. Когда кредиты заканчиваются, Cloud AI ставится на паузу до продления или пополнения.
  • Приватность: выделенный текст идет в backend Lexipane, затем к провайдеру. Desktop-приложение не содержит мастер-ключ OpenAI.

OpenAI со своим ключом

  • Для кого: личное использование в рамках ваших лимитов OpenAI.
  • Оплата: запросы идут напрямую с компьютера в OpenAI и не тратят кредиты Lexipane Cloud.
  • Связь: ваша сеть должна открывать OpenAI. Если OpenAI недоступен в регионе, используйте VPN или переключитесь на Without AI.

Without AI

  • Для кого: быстрый перевод, offline-словари и базовые Anki-карточки.
  • Online fallback: при включении может использовать неофициальный endpoint Google Translate.
  • Только offline: отключает web-запросы, но качество зависит от установленных словарных паков.

Правило кредитов

Кредиты Lexipane Cloud проверяются на сервере. Desktop-приложение не хранит мастер-ключ OpenAI и не создает OpenAI API keys само. Если backend или OpenAI недоступны, Lexipane показывает понятное сообщение: можно повторить позже, включить VPN, перейти на свой ключ или использовать Without AI.
