AI modes in Lexipane

Pro OpenAI Without AI Provider settings · credits · local fallback

Lexipane has two translation modes. Use OpenAI when you want grammar, nuance and context explanations. Use Without AI when you want a free fallback that starts with installed dictionaries and then uses the configured translator endpoint.

Provider settings in the desktop app.

OpenAI mode

  • Best for: sentence explanations, grammar, idioms, phrasal verbs and context-aware cards.
  • Requires: Pro access plus either Lexipane Cloud credits or your own OpenAI API key.
  • Privacy: selected text is sent to the chosen AI provider only when you ask Lexipane to explain or generate content.

Without AI mode

  • Best for: quick translation, offline dictionary lookups and basic Anki cards.
  • Online fallback: can use Google Translate unofficial endpoint when enabled.
  • Offline-only: disables web requests, but quality depends entirely on installed dictionary packs.

Credit rule

Lexipane Cloud credits must be enforced on the server. The desktop app never stores the master OpenAI key and never creates OpenAI keys by itself. When credits run out, the app should show a top-up prompt or let you switch to your own key.
