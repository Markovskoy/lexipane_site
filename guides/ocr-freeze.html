OCR and Game Freeze

Windows Experimental Pro Hardware · OCR · fair use
Hardware recommendation: Windows 10/11, 16 GB RAM, NVIDIA GPU for local OCR acceleration. CPU mode works, but recognition is slower. Game Freeze is experimental and should not be used with online games, anti-cheat games, launchers or browsers.

On macOS, Lexipane hides OCR engine settings because the default capture flow is already configured. On Windows, OCR settings focus on one task: install/check local OCR acceleration for Pause + Read and Game Freeze.

Runtime settings before the OCR cleanup pass.

Recommended setup

  1. Use the default OCR provider for normal screen capture.
  2. For Game Freeze, enable the Pro feature and run the hardware check.
  3. If NVIDIA is detected, install GPU OCR and restart Lexipane.
  4. Keep allowlist/denylist conservative. Do not pause online or protected games.

Why this is Pro

Game Freeze touches running processes and requires extra guardrails, support and testing. Lexipane should keep the user-facing control simple: enable the feature, install optional OCR acceleration, and show a clear warning when hardware is not suitable.
