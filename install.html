Install Lexipane

Download and set up Lexipane on your computer. Choose your platform below.

Windows

Windows 10 / 11 · 64-bit

  1. 1
    Download the installer — нажмите кнопку ниже, сохраните файл Lexipane.exe.
  2. 2
    Запустите установщик. Если Windows SmartScreen показывает предупреждение — нажмите «Подробнее» → «Выполнить в любом случае».
  3. 3
    Следуйте шагам установщика. После — запустите Lexipane из меню Пуск или рабочего стола.
  4. 4
    При первом запуске пройдите быструю настройку — выберите языки и режим перевода.

macOS

macOS 13 Ventura и новее

  1. 1
    Скачайте DMG — нажмите кнопку ниже.
  2. 2
    Откройте скачанный .dmg файл. Перетащите Lexipane.app в папку Applications.
  3. 3
    В открытом DMG запустите «Install Lexipane.command» — он автоматически уберёт системную блокировку и откроет приложение.
    Если появится предупреждение Terminal — нажмите «Открыть».
  4. 4
    macOS попросит три разрешения — нажмите «OK» на каждое:
    • Специальные возможности (Accessibility)
    • Мониторинг ввода (Input Monitoring)
    • Запись экрана (Screen Recording)
    Разрешения запрашиваются один раз — при обновлениях повторно не нужны.
